# 🩸 BloodConnect – Blood Donation Web App

## 📌 Project Overview

BloodConnect is a frontend-based web application designed to connect blood donors with people in need.
It helps users request blood, find nearby donors, and understand blood donation awareness.

---

## 🚀 Features

### 🔐 Authentication

* User Login & Registration pages
* Simple form validation

### 🏠 Home Page

* Hero section with call-to-action buttons
* Emergency blood request section
* About blood donation
* Statistics section
* Process of donation
* Testimonials
* Contact form (EmailJS integrated)

### 🩸 Blood Request

* Request blood with patient details
* Select blood group and urgency level
* Auto alert message on submission
* Filter and display nearby donors

### ❤️ Donate Page

* Register as a donor
* Provide blood group, location, and contact
* Eligibility and availability options

### 📚 Awareness Page

* Precautions after donation
* Eligibility criteria (who can / cannot donate)
* Benefits of blood donation
* FAQ section
